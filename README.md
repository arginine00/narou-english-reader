# NarouEN Reader

> 小説家になろうの小説をリアルタイム英語翻訳・音声読み上げするアプリ / Chrome拡張

---

## 機能

| 機能 | 説明 |
|------|------|
| **英語翻訳表示** | なろう小説の本文を段落ごとに英訳して表示 |
| **TTS 読み上げ** | 英語テキストを音声で読み上げ（速度調整可能）|
| **単語ハイライト** | 読み上げ中の単語をリアルタイムでハイライト |
| **1文モード** | 1文再生して停止 → ボタンで次の文へ |
| **連続モード** | 最後の文まで自動で読み進める |
| **単語タップ → 訳** | 英単語を1クリックで日本語訳を表示 |
| **文置き換え** | ダブルクリック / 長押しで英文 ↔ 日本語に置き換え |
| **文節対訳** | 日本語表示中にダブルクリックで文節ごとの対訳カードを表示 |
| **翻訳キャッシュ** | 一度翻訳したテキストはローカルに保存してAPI消費を節約 |

## プロジェクト構成

```
narou-english-reader/
├── web-extension/          # Chrome / Firefox 拡張機能
│   ├── manifest.json       # Manifest V3
│   ├── background.js       # Service Worker（翻訳API呼び出し）
│   ├── content.js          # なろうページ検知・本文抽出
│   ├── popup.html          # 設定画面（APIキー入力など）
│   └── src/
│       ├── reader.html     # リーダー UI（スタンドアロンでも動作）
│       ├── reader.js       # コアロジック（TTS・インタラクション）
│       ├── data.js         # サンプルデータ
│       └── dict.js         # 単語辞書
├── android-app/
│   └── README.md           # Android実装ガイド（Kotlin + Jetpack Compose）
└── README.md               # このファイル
```

## クイックスタート（Web 拡張）

### 1. Chrome に読み込む

```
1. chrome://extensions を開く
2. 「デベロッパーモード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」
4. web-extension/ フォルダを選択
```

### 2. DeepL API キーを設定

```
1. https://www.deepl.com/pro-api で無料キーを取得
2. 拡張機能のアイコンをクリック → Settings
3. API キーを入力して保存
```

### 3. なろうで読む

```
https://ncode.syosetu.com/ の小説ページを開くと自動的にリーダーが表示される
```

## 操作方法

### 英文表示中

| 操作 | 動作 |
|------|------|
| 単語を **1クリック** | 単語の日本語訳を下パネルに表示 |
| 単語を **ダブルクリック** | 英文 → 日本語に置き換え |
| 単語を **長押し（500ms）** | 英文 → 日本語に置き換え |

### 日本語表示中

| 操作 | 動作 |
|------|------|
| **ダブルクリック** / **長押し** | 文節ごとの英語↔日本語対訳カードを表示 |
| **← 英文に戻す** | 英文に戻す |

### 再生コントロール

| 操作 | 動作 |
|------|------|
| **1文のみ** モード | 1文読み上げ → 停止、次の ▶ で続きの文 |
| **連続** モード | 最後の文まで自動で読み進める |
| **◀ / ▶** ボタン | 文を移動（再生中は再生を維持）|
| **速度セレクト** | 0.75× / 1.0× / 1.25× / 1.5× |

## 翻訳 API

| API | 無料枠 | 備考 |
|-----|--------|------|
| **DeepL Free** | 500,000 字/月 | 文学テキストの品質が高い（推奨）|
| Google Cloud Translation v3 | $300 クレジット | 汎用 |

## 利用規約・注意事項

- なろうのスクレイピングは **個人・非商用利用** の範囲でのみ行うこと
- 翻訳テキストの再配布・公開は著作権法上 NG
- サーバー負荷を避けるため、リクエスト間隔に 300ms 以上の遅延を設けること
- `onboundary` による単語ハイライトは **Chrome / Firefox** で動作、Safari は非対応
